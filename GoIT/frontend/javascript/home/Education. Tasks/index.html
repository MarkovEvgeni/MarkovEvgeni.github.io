Javascript Modules 01-02

Основы Javascript. Переменные. Типы данных. Основные операторы. Условные операторы "if" и "?". Массивы. Циклы. Объекты. Фнукции. Глобальный объект "Window". Стандартные методы массивов, строк и чисел.

Задание к модулю

Написать функцию pow, аналогичную Math.pow, которая должна возводить указанное число в указанную степень. Указать число и степень пользователь должен через команду prompt. Результат выполнения функции вывести в консоль. Функция должна работать с целыми числами, большими, меньшими, и равными нулю. Бесконечности обрабатывать не обязательно.

Создать программу, которая будет выполнять следующие действия:

  • При помощи команды prompt циклом заполнять массив, в котором будет список из пяти любых имен.
  • При помощии команды prompt выводить сообщение с просьбой ввести имя пользователя.
  • Введенное имя циклом сравнивать с именами в массиве.
  • Если нет совпадения, то есть введенное имя пользователя не существует в массиве - выдавать с помощью alert сообщение об ошибке.
  • Если есть совпадение - выводить сообщение "Андрей, вы успешно вошли". Вместо "Андрей" должно быть имя текущего пользователя.

Исполнение

Калькулятор степеней. Готовая страница.

Калькулятор степеней. Репозиторий.

Список пользователей. Готовая страница.

Javascript Modules 05-06

События Javascript. Функции "setTimeout" и "setInterval". Конструкторы. Оператор "New". Методы "call" и "apply".

Задание к модулю

Требуется создать вот такой таймер Stop Watch, который должен обладать следующим функционалом:

  • Кнопка Start запускает таймер, так же нужно выводить милисекунды. После запуска кнопка меняется на кнопку Pause, которая может приостанавливать таймер.
  • Кнопка Clear останавливает и обнуляет таймер.
  • Дизайн свой, используя Pure или Bootstrap.

Бонусное задание. Создать более сложный таймер, который должен обладать следующим функционалом:

  • Кнопка Start начинает отсчет с нуля. При работе таймер выводит миллисекунды. Кнопка Start после запуска меняется на Stop.
  • Кнопка Split засекает текущий отсчет и выводит его под таймером с меткой Split, при этом не останавливая работу таймера.
  • Кнопка Stop - останавливает текущий отсчет. Время, которое засечено при помощи этой кнопки, выводится под таймером с пометкой Stop.
  • Кнопка Reset - обнуляет таймер, удаляя все сохраненные отметки Split и Stop.

Исполнение

Оба таймера. Готовая страница.

Оба таймера. Репозиторий.
